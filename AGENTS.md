# 个人代办事项应用 AI 开发指令

## 项目概述
这是一个轻量级个人代办事项管理应用，旨在帮助用户养成良好的任务管理习惯，通过简洁直观的界面实现任务的分类管理、进度跟踪和数据统计。使用 React + TypeScript + Vite 开发。

## 开发规范
- 使用 TypeScript，确保类型安全
- 组件使用函数式组件 + Hooks
- 使用 Tailwind CSS 编写样式
- 所有数据存储在 LocalStorage
- 遵循三栏式布局设计（左侧边栏 + 顶部操作区 + 主内容区）

## 代码风格
- 使用 ESLint 和 Prettier
- 组件名使用 PascalCase
- 函数名使用 camelCase
- 常量使用 UPPER_SNAKE_CASE
- 类型定义使用 TypeScript 接口

## 功能实现优先级
### MVP 版本（优先实现）
1. 基础任务创建功能（标题输入、分类选择）
2. 任务完成状态切换
3. 分类筛选功能
4. 基础统计数据展示（总任务、待完成、已完成、完成率）
5. 本地数据存储

### 后续版本
1. 任务优先级设置
2. 任务截止日期设置
3. 任务搜索功能
4. 任务编辑和删除功能
5. 自定义分类创建

## 测试要求
- 每个功能完成后手动测试
- 确保数据正确存储和读取
- 测试各种边界情况
- 验证统计数据计算准确性
